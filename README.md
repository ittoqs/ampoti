# 📦 Ampoti Archiver

[![Kotlin](https://img.shields.io/badge/Kotlin-Native-purple.svg)](https://kotlinlang.org/)
[![Platform](https://img.shields.io/badge/Platform-Android_11%2B-green.svg)](https://developer.android.com)

A native Android File Manager application focused specifically on advanced archive management. Easily compress, extract, and navigate through multiple archive formats directly from your mobile device.

---

## ✨ Features

* **📁 Local File Browsing:** Seamlessly navigate and manage files on your device storage.
* **🗜️ Archive Management:** Complete support for creating and extracting major archive types including **ZIP, RAR, 7Z, TAR, GZ**, and more.
* **🔒 Android 11+ Ready:** Fully optimized to comply with modern Android storage policies using the `MANAGE_EXTERNAL_STORAGE` permission for unrestricted file access.

---

## 🛠️ Tech Stack

This project is built using powerful, standard, and modern open-source libraries:

* **[Kotlin](https://kotlinlang.org/)** - Modern and expressive programming language for Android development.
* **[AndroidX](https://developer.android.com/jetpack/androidx)** - Jetpack libraries for reliable, backward-compatible UI and component behavior.
* **[Apache Commons Compress](https://commons.apache.org/proper/commons-compress/)** - Robust API for working with AR, ARJ, CPIO, DUMP, TAR, ZIP, BZIP2, GZIP, PACK200, XZ, and DEFLATE formats.
* **[Junrar](https://github.com/junrar/junrar)** - Pure Java library for extracting RAR archives.
