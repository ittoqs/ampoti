Ampoti Archiver is a native mobile file manager focused specifically on advanced archive management. Easily compress, extract, and navigate through multiple archive formats directly from your Android device.

---

## ✨ Features

* **📁 Local File Browsing:** Seamlessly navigate and manage files on your device storage.
* **🗜️ Comprehensive Archive Support:** Complete functionality for creating and extracting major archive types including **ZIP, RAR, 7Z, TAR, GZ**, and more.
* **🔒 Android 11+ Optimized:** Fully adapted to modern Android storage policies utilizing `MANAGE_EXTERNAL_STORAGE` permission for high-performance, unrestricted file access.

---

## 🛠️ Tech Stack

This project is built using powerful, standard, and modern open-source libraries:

* **[Kotlin](https://kotlinlang.org/)** - Modern and expressive programming language for Android development.
* **[AndroidX](https://developer.android.com/jetpack/androidx)** - Jetpack libraries for reliable, backward-compatible UI and component behavior.
* **[Apache Commons Compress](https://commons.apache.org/proper/commons-compress/)** - Robust API for working with AR, ARJ, CPIO, DUMP, TAR, ZIP, BZIP2, GZIP, PACK200, XZ, and DEFLATE formats.
* **[Junrar](https://github.com/junrar/junrar)** - Pure Java library for extracting RAR archives.
